### **Object-Oriented Ontology**

The version under audit is the one in which the real object is constitutively foreclosed from all relation \- Harman's defining thesis.[^32]

Harman begins with a correct insight: no interaction exhausts what an entity is. The hammer exceeds this moment of hammering; the organism exceeds this moment of measurement. Reduction to current relational profile is a falsification of experience. Compositional immanence inherits the insight without reservation: relational inexhaustibility is non-negotiable.

The move from relational inexhaustibility to constitutive withdrawal \- the thesis that the real object is sealed from all relational contact \- requires a distinction (withdrawn/sensual) maintained across every encounter and every context. A distinction maintained by energetic favourability \- the diamond's lattice \- persists without throughput, but only against perturbations below its stability threshold. Constitutive withdrawal claims the stronger condition: stability against all relational contact. That condition requires active regeneration under arbitrary perturbation, which is a throughput-dependent operation \- yet the framework specifies no throughput and provides no cessation-test for the withdrawn/sensual boundary.[^33]

The consequence falls at genesis: if real objects are constitutively sealed from relational installation, no encounter can precipitate a new constraint-identity. Vicarious causation rearranges sensual profiles; it cannot install the closure that would constitute a new regime. The genesis problem (1.1) has no purchase in a framework whose real objects are inaccessible to the relational structures from which closures precipitate.

Compositional immanence relocates the surplus Harman correctly identifies: it is ahead of the object, in the metastable field's undischarged potentials, not behind it in concealment. The marble block exceeds the sculptor's chisel not because a withdrawn core hides from relation, but because the block's crystalline microstructure carries thermodynamic potentials \- fracture planes, stress distributions, grain boundaries \- that no single encounter can discharge. Inexhaustibility is funded by metastable surplus, not by ontological seclusion.
